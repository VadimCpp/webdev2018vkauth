# webdev2018vkauth

Данный проект закрыт

## Getting Started

**NOTE!** localhost at port 80 should be available

```bash
  git clone https://github.com/VadimCpp/webdev2018vkauth webdev2018vkauth
  cd webdev2018vkauth
  yarn install
  yarn run dev
```
